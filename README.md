# 🌾 Smart Agriculture AI

An AI-powered system that helps farmers and users make better agricultural decisions.

## 🚀 Features

- 🌱 Crop Recommendation based on soil nutrients and environmental conditions
- 💧 Irrigation Advice using smart rule-based logic
- 📊 Yield Prediction using Machine Learning
- 📈 Data Visualization with interactive charts

## 🧠 Technologies Used

- Python
- Scikit-learn
- Streamlit
- Pandas

## 🌍 Live Application

👉 https://smart-agri-ai.streamlit.app

## 📌 How it Works

1. User enters soil parameters (N, P, K, pH)
2. Adds environmental data (temperature, humidity, rainfall)
3. Model predicts best crop
4. System provides irrigation advice and insights

## 💼 Project Highlights

- Built using Machine Learning algorithms
- Deployed as a live web application
- Multi-page interactive UI
- Real-world problem solving
